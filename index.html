const calcularAreaTriangulo = (base, altura) => {

  const area = (base * altura) / 2;

  return area;

};


const baseTriangulo = 5;

const alturaTriangulo = 8;

const areaTriangulo = calcularAreaTriangulo(baseTriangulo, alturaTriangulo);


console.log(`El área del triángulo con base ${baseTriangulo} y altura ${alturaTriangulo} es: ${areaTriangulo}`);


  • En este código, creamos una función llamada calcularAreaTriangulo que toma dos parámetros: base y altura. Dentro de la función, calculamos el área del triángulo multiplicando la base por la altura y dividiendo el resultado por 2. Luego, retornamos el valor del área.

  • Después, definimos las variables baseTriangulo y alturaTriangulo con los valores del triángulo que deseamos calcular el área.

  • Luego, llamamos a la función calcularAreaTriangulo pasando las variables baseTriangulo y alturaTriangulo como argumentos, y almacenamos el resultado en la variable areaTriangulo.

  • Finalmente, utilizamos console.log() para imprimir en la consola el resultado del cálculo, mostrando el valor de la base, la altura y el área del triángulo.

<!DOCTYPE html>

<html>

<head>

  <title>Calculadora</title>

  <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">

  <style>

    .calculator {

    }

  </style>

</head>

<body>

  <div class="container">

    <div class="calculator">

      <div class="result"></div>

      <div class="buttons">

        <button class="btn btn-secondary">1</button>

        <button class="btn btn-secondary">2</button>

        <button class="btn btn-secondary">3</button>

        <button class="btn btn-secondary">+</button>

        <button class="btn btn-secondary">4</button>

        <button class="btn btn-secondary">5</button>

        <button class="btn btn-secondary">6</button>

        <button class="btn btn-secondary">-</button>

        <button class="btn btn-secondary">7</button>

        <button class="btn btn-secondary">8</button>

        <button class="btn btn-secondary">9</button>

        <button class="btn btn-secondary">*</button>

        <button class="btn btn-secondary">0</button>

        <button class="btn btn-secondary">.</button>

        <button class="btn btn-secondary">=</button>

        <button class="btn btn-secondary">/</button>

        <button class="btn btn-secondary">C</button>

      </div>

    </div>

  </div>

  

  <script>

    const resultElement = document.querySelector('.result');

    const buttons = document.querySelectorAll('.buttons button');

    let result = '';


    buttons.forEach(button => {

      button.addEventListener('click', () => {

        const buttonText = button.textContent;

        

        if (buttonText === '=') {

          try {

            result = eval(result);

            resultElement.textContent = result;

          } catch (error) {

            resultElement.textContent = 'Error';

          }

        } else if (buttonText === 'C') {

          result = '';

          resultElement.textContent = '';

        } else {

          result += buttonText;

          resultElement.textContent = result;

        }

      });

    });

  </script>

</body>

</html>


  • En este ejercicios, utilizamos HTML para crear la estructura básica de la calculadora. Utilizamos la biblioteca Bootstrap para aplicar estilos y dar formato a los elementos de la calculadora.

  • En el CSS, definimos algunos estilos personalizados para el contenedor de la calculadora, el resultado y los botones.

  • En el JavaScript, hacemos uso de document.querySelector para seleccionar el elemento de resultado y document.querySelectorAll para seleccionar todos los botones. Luego, añadimos un event listener a cada botón para capturar el evento de clic.

  • Dentro del event listener, verificamos qué botón se ha clicado. Si el botón es el de igual (=), evaluamos la expresión utilizando la función eval de JavaScript y mostramos el resultado en el elemento de resultado. Si el botón es el de borrar (C), reiniciamos la variable result y borramos el contenido del elemento de resultado. Si el botón es cualquier otro número o operador, concatenamos el texto del botón a la variable result y mostramos la expresión actualizada en el elemento de resultado.

  • Este es solo un ejemplo básico de calculadora utilizando HTML, CSS y JavaScript junto con la biblioteca Bootstrap. Puedes personalizar y extender este código según tus necesidades.

<head>

    body {

    }

    .task-list {

      border: 1px solid #ccc;

      list-style-type: none;

      padding: 0;

    }

    .task-list li {

      display: flex;

      align-items: center;

      margin-bottom: 5px;

    }

    .task-list li input[type="checkbox"] {

      margin-right: 10px;

    }

    .task-list button {

      background-color: #4CAF50;

      color: white;

      border: none;

      padding: 8px 16px;

      text-align: center;

      text-decoration: none;

      display: inline-block;

      font-size: 14px;

      margin-top: 10px;

      cursor: pointer;

      border-radius: 5px;

    }

  </style>

</head>

<body>

  <div class="task-list">

    <h2>Lista de Tareas</h2>

    <input type="text" id="taskInput" placeholder="Agregar una tarea">

    <ul id="taskList"></ul>

    <button id="addTaskBtn">Agregar Tarea</button>

  </div>


  <script>

    const taskInput = document.querySelector('#taskInput');

    const taskList = document.querySelector('#taskList');

    const addTaskBtn = document.querySelector('#addTaskBtn');


    addTaskBtn.addEventListener('click', () => {

      const taskText = taskInput.value;

      if (taskText.trim() !== '') {

        const taskItem = document.createElement('li');

        const checkBox = document.createElement('input');

        checkBox.type = 'checkbox';

        const taskLabel = document.createElement('label');

        taskLabel.textContent = taskText;

        taskItem.appendChild(checkBox);

        taskItem.appendChild(taskLabel);

        taskList.appendChild(taskItem);

        taskInput.value = '';

      }

    });


    taskList.addEventListener('change', (event) => {

      const taskItem = event.target.parentElement;

      if (event.target.checked) {

        taskItem.classList.add('completed');

      } else {

        taskItem.classList.remove('completed');

      }

    });

  </script>

</body>

</html>


  • En este ejercicios, creamos una lista de tareas básica utilizando HTML, CSS y JavaScript. El usuario puede ingresar una tarea en un campo de texto y agregarla a la lista haciendo clic en el botón "Agregar Tarea". Cada tarea se muestra como un ítem de lista con una casilla de verificación.

  • El CSS se utiliza para dar formato a la lista de tareas y a los elementos individuales. Se utilizan selectores de clase y de etiqueta para aplicar estilos a los diferentes elementos.

  • El JavaScript se encarga de capturar el evento de clic en el botón "Agregar Tarea". Cuando se hace clic en el botón, se recupera el texto de la tarea ingresada por el usuario y se crea un nuevo ítem de lista con una casilla de verificación y una etiqueta. El ítem se agrega a la lista de tareas y el campo de texto se vacía.

  • También se agrega un event listener para el evento "change" en la lista de tareas. Cuando se marca o desmarca una casilla de verificación, se agrega o se elimina la clase "completed" del ítem correspondiente, lo que permite aplicar estilos visuales para indicar si la tarea ha sido completada o no.

  • Este es solo un ejemplo básico de una lista de tareas interactiva utilizando HTML, CSS y JavaScript. Puedes personalizar y ampliar este código según tus necesidades y agregar funcionalidades adicionales, como eliminar tareas o guardar el estado de las tareas en el almacenamiento local.

Desarrollo  FrontEnd con ECMAScript

<!DOCTYPE html>

<html>

<head>

  <title>Operación de Suma</title>

  <script type="module">

    const sum = (a, b) => {

      return a + b;

    };


    const handleSum = () => {

      const num1 = parseFloat(document.getElementById('num1').value);

      const num2 = parseFloat(document.getElementById('num2').value);

      const result = sum(num1, num2);

      document.getElementById('result').textContent = `El resultado es: ${result}`;

    };

  </script>

</head>

<body>

  <h1>Operación de Suma</h1>

  <label for="num1">Número 1:</label>

  <input type="number" id="num1" placeholder="Ingrese el primer número">

  <br>

  <label for="num2">Número 2:</label>

  <input type="number" id="num2" placeholder="Ingrese el segundo número">

  <br>

  <button onclick="handleSum()">Sumar</button>

  <br>

  <div id="result"></div>

</body>

</html>


  • En este ejemplo, utilizamos ECMAScript para definir una función de suma llamada sum que toma dos parámetros a y b y devuelve la suma de ambos.

  • En el HTML, creamos un formulario simple con dos campos de entrada de números y un botón "Sumar". También tenemos un elemento <div> vacío con el id "result" donde mostraremos el resultado de la suma.

  • En el JavaScript, utilizamos document.getElementById para obtener los valores ingresados en los campos de entrada de números. Luego, llamamos a la función sum con los valores obtenidos y almacenamos el resultado en la variable result. Finalmente, actualizamos el contenido del elemento con id "result" para mostrar el resultado de la suma al usuario.

  • Puedes probar este ejemplo ingresando dos números en los campos de entrada y haciendo clic en el botón "Sumar". El resultado de la suma se mostrará debajo del botón.

  • Este es solo un ejemplo básico de cómo utilizar ECMAScript en el desarrollo frontend para realizar una operación simple en el navegador. Puedes personalizar y expandir este código según tus necesidades y agregar funcionalidades adicionales utilizando las características y sintaxis avanzadas de ECMAScript.

Introducción al Desarrollo con React

import React, { useState } from 'react';


function Counter() {

  const [count, setCount] = useState(0);


  const increment = () => {

    setCount(count + 1);

  };


  const decrement = () => {

    setCount(count - 1);

  };


  return (

    <div>

      <h1>Contador</h1>

      <p>Valor actual: {count}</p>

      <button onClick={increment}>Incrementar</button>

      <button onClick={decrement}>Decrementar</button>

    </div>

  );

}


export default Counter;


  • En este ejemplo, utilizamos la biblioteca React para crear un componente funcional llamado Counter. El componente utiliza el hook useState para gestionar el estado interno de un contador.

  • Dentro del componente, utilizamos la destructuración de asignación para obtener la variable count y la función setCount del hook useState. Inicializamos el contador en 0.

  • Luego, definimos las funciones increment y decrement que actualizan el valor del contador utilizando la función setCount y el estado actual de count.

  • En el método render del componente, utilizamos JSX para definir la estructura y la interfaz de usuario. Mostramos el valor actual del contador utilizando la variable count. También añadimos dos botones con los eventos onClick que llaman a las funciones increment y decrement respectivamente.

  • Finalmente, exportamos el componente Counter para que pueda ser utilizado en otros componentes o en la aplicación principal de React.

  • Este es solo un ejemplo básico que muestra cómo crear un componente simple de contador utilizando React. Puedes utilizar este código como punto de partida y ampliarlo o personalizarlo según tus necesidades. React ofrece muchas más características y opciones para el desarrollo frontend, como el manejo de formularios, la gestión del estado global con Context API o Redux, y la integración con APIs externas.
